Que es SQLAlchemy?

Es una biblioteca de Python para conectarse a bases de datos SQL

Ofrece:
-Conexion simple (create_engine)
-ORM (No lo vemos por ahora) (Object  Relational-Mapping)
-Compatible con Pandas

Que necesito correr para qu efuncione?
pip install sqlalchemy pandas mysql-connector-python
